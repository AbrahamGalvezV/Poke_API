# Pokémon Guesser 🎮⚡

Una aplicación desarrollada con **React + TypeScript** que desafía al jugador a adivinar el nombre de un Pokémon a partir de su silueta. El proyecto consume datos de la **PokéAPI** y permite jugar utilizando distintas generaciones de Pokémon.

## ✨ Características

- 🎲 Selección aleatoria de Pokémon.
- 👤 Introducción manual del nombre para comprobar la respuesta.
- 👻 Imagen oculta mediante silueta hasta responder.
- ✅ Comprobación de respuestas correctas e incorrectas.
- 🔄 Generación instantánea de un nuevo Pokémon.
- 🌍 Consumo de datos desde PokéAPI.
- 📚 Selección de generación Pokémon.
- 🎨 Interfaz inspirada en una Pokédex futurista.

## 🚀 Instalación

Clona el repositorio:

```bash
git clone https://github.com/tu-usuario/pokemon-guesser.git
```

Accede al directorio:

```bash
cd pokemon-guesser
```

Instala las dependencias:

```bash
npm install
```

Inicia el servidor de desarrollo:

```bash
npm run dev
```

La aplicación estará disponible en:

```txt
http://localhost:5173
```

---

## 🎮 Cómo jugar

1. Selecciona una generación desde el menú hamburguesa.
2. Observa la silueta del Pokémon.
3. Escribe el nombre que crees que corresponde.
4. Pulsa **Comprobar**.
5. Descubre si tu respuesta es correcta.
6. Utiliza **Volver a intentar** para continuar jugando.
